# Stock Price Prediction using LSTM & ANN

This project predicts stock prices using Artificial Neural Networks (ANN) and Long Short-Term Memory (LSTM) models built with Python and TensorFlow/Keras.

## Features
- Stock dataset preprocessing
- Data normalization using MinMaxScaler
- ANN and LSTM model training
- Mean Squared Error (MSE) comparison
- Stock price visualization using Matplotlib
- GUI built with Tkinter

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
- Tkinter

## Project Goal
The goal of this project is to analyze historical stock market data and compare ANN and LSTM models for stock price prediction.
